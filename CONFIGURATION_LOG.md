## 📊 الخدمات المشغلة تلقائياً

```
عند تشغيل docker-compose up -d:

1. 🛡️ WAF (جدار الحماية)
   └─ البرت: 127.0.0.1:5000
   └─ يفتش جميع الطلبات
   └─ يرسل تنبيهات Telegram

2. 💻 PHP Backend
   └─ المنفذ: 127.0.0.1:8080
   └─ لا يمكن الوصول مباشرة من الإنترنت
   └─ يعالج الطلبات الآمنة فقط

3. 🌐 Nginx (البوابة الأمامية)
   └─ المنفذ: 0.0.0.0:80 (مُعاد إلى 8001 للاختبار المحلي)
   └─ توجيه إلى WAF أولاً
   └─ حماية SSL وضغط Gzip

4. 📊 Docker Network
   └─ الاسم: kreen-network
   └─ يربط جميع الخدمات
```

## 🚀 كيفية البدء الآن

### الطريقة الأولى: السكريبت السريع (الأسهل)

```bash
cd /home/kali/Videos/kreen
bash quick-start.sh
```

### الطريقة الثانية: اليدوية

```bash
cd /home/kali/Videos/kreen

# 1. تحديث .env بـ Telegram Token و Chat ID
nano .env

# 2. البناء والتشغيل
docker-compose down 2>/dev/null || true
docker build -t kreen-app .
docker-compose up -d

# 3. فحص
docker-compose ps
```

## 🚀 البدء السريع (5 دقائق فقط)

### الخطوة 1: النسخ والإعدادات
```bash
# انتقل إلى مجلد المشروع
cd kreen

# نسخ الإعدادات
cp .env.example .env

# تثبيت المكتبات
composer install
```

### الخطوة 2: الأذونات
```bash
# إعطاء أذونات التخزين
chmod 755 uploads/ logs/ sessions/
chmod 666 kreen.db

# التحقق من الإعدادات
cat .env | grep TELEGRAM
```

### الخطوة 3: التشغيل
```bash
# تشغيل خادم الويب
php -S localhost:8080

# أو استخدم Docker
docker-compose up -d
```

### الخطوة 4: الدخول
```
URL: http://localhost:8080
البريد: admin@kreen.com
كلمة المرور: admin123 (بعد التسجيل)
```
